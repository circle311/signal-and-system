\section{项目目标}

本项目使用一段真实音频信号完成信号与系统中的基本时域变换和频域分析。
音频采样序列记为 $x[n]$，通过插值方法构造其连续时间近似 $x(t)$。
随后分别研究时间反转、时间尺度变换、傅里叶频谱、幅度谱与相位谱的逆变换，
以及频域理想低通滤波。整个实验使用 MATLAB 完成，生成的音频、波形图和频谱图
均可由源码重新得到。

\section{输入音频与 MATLAB 环境}

本项目的实现语言为 MATLAB。输入音频文件为
\texttt{audio/input/original.wav}。该音频的基本参数如下：

\begin{itemize}[nosep]
  \item 格式：WAV，PCM signed 16-bit little-endian。
  \item 采样率：16000 Hz。
  \item 声道数：单声道。
  \item 时长：7.000 秒。
  \item 预处理：读入后进行幅度归一化，避免后续写出音频时削波。
\end{itemize}

该音频长度适中。一方面，7 秒的长度足以体现较丰富的频谱内容；另一方面，
它又不会使波形图和 FFT 频谱图过于拥挤，适合作为本项目的输入信号。

\section{插值与连续时间近似}

MATLAB 使用 \texttt{audioread} 读取离散音频样本 $x[n]$。由于计算机无法
直接表示真正连续的 $x(t)$，本项目采用线性插值作为连续时间信号的近似。
具体来说，相邻采样点之间用直线连接，从而构造可绘制、可分析的 $x(t)$。
图~\ref{fig:interp} 展示了短时间窗口内的离散样本和线性插值曲线。

\section{时域变换}

根据项目要求，本文生成并分析以下三个时域变换信号：

\begin{itemize}
  \item $x(-t)$：时间反转。对有限长音频而言，可通过反转采样序列实现，
  听感上相当于倒放原始音频。
  \item $x(2t)$：时间压缩。时间轴压缩为原来的一半，因此输出时长约为
  3.5 秒。
  \item $x(t/2)$：时间扩展。时间轴扩展为原来的两倍，因此输出时长约为
  14 秒。
\end{itemize}

从听感上看，$x(-t)$ 保留了原音频的能量分布但播放顺序相反；$x(2t)$
变化更快，时长更短；$x(t/2)$ 变化更慢，时长更长。

\section{傅里叶变换与频谱分析}

本项目使用 FFT 计算各信号的频谱，并使用 \texttt{fftshift} 将负频率部分
移动到图像左侧、正频率部分移动到图像右侧。幅度谱经过归一化后以 dB 为单位绘制，
这样可以更清楚地比较频谱形状，而不是只比较绝对幅度。

对于时间反转 $x(-t)$，幅度谱整体上与原信号相近，但相位发生变化。
对于 $x(2t)$，时域压缩对应频域扩展；对于 $x(t/2)$，时域扩展对应频域压缩。
这与傅里叶变换的时间尺度性质一致：

\[
x(at) \Longleftrightarrow \frac{1}{|a|}X\left(j\frac{\omega}{a}\right).
\]

\section{幅度谱与相位谱的逆变换}

设 $X(j\omega)$ 为 $x(t)$ 的傅里叶变换。项目要求分别观察幅度谱和相位谱
各自对信号重构的影响，因此本文比较以下两个信号：

\[
\mathcal{F}^{-1}\{|X(j\omega)|\}
\]

和

\[
\mathcal{F}^{-1}\{e^{j\angle X(j\omega)}\}.
\]

幅度谱逆变换保留了原信号的频率能量分布，但丢弃了相位信息；相位谱逆变换
保留了相位关系，但丢弃了原始幅度分布。实验结果表明，两种重构波形都与原始
音频有明显差异，说明幅度和相位对于完整恢复音频信号都很重要。特别是对于语音
或音乐信号，相位关系会影响波形的时域结构，而幅度谱决定各频率成分的强弱。

\section{频域理想低通滤波}

本项目在频域中对 $x(t)$ 实现理想低通滤波。输入音频采样率为 16000 Hz，
因此奈奎斯特频率为 8000 Hz。本文选择截止频率为 3000 Hz。理想低通滤波器的
频率响应为

\[
H(f)=
\begin{cases}
1, & |f|\leq 3000\ \text{Hz},\\
0, & |f|>3000\ \text{Hz}.
\end{cases}
\]

该滤波器保留低频和中频成分，同时去除 3000 Hz 以上的高频成分。实际处理时，
MATLAB 先对信号作 FFT，再用二值频域掩膜乘以频谱，最后通过 IFFT 得到滤波后
的时域信号。滤波后的波形和频谱如下图所示。

\section{第二部分：OTFS 论文阅读与理解}

本文选择的论文为 P. Raviteja、Khoa T. Phan、Yi Hong 和 Emanuele Viterbo
的 \emph{Interference Cancellation and Iterative Detection for Orthogonal
Time Frequency Space Modulation}。该论文发表于 2018 年的
\emph{IEEE Transactions on Wireless Communications}，公开预印本可见
\url{https://arxiv.org/abs/1802.05242}。

这篇论文研究 OTFS 调制在延迟-多普勒信道中的检测问题。在高速移动场景中，
无线信道会随时间快速变化，多普勒扩展会破坏 OFDM 子载波之间的正交性，
从而产生明显的载波间干扰。OTFS 的基本思想是将信息符号放置在延迟-多普勒域，
再通过二维变换映射到常规时频域进行发送。

论文的主要内容可以概括为以下几点。首先，作者推导了 OTFS 调制和解调在
延迟-多普勒信道中的输入输出关系。该关系表明，接收符号受到的干扰主要来自
延迟-多普勒域中有限数量的相邻位置，而不是在整个时频平面上无序扩散。其次，
作者分析了 OTFS 中的干扰结构，并提出低复杂度消息传递检测算法。该算法利用
信道稀疏性，通过迭代方式估计符号并抵消干扰。

从信号与系统角度理解，OTFS 的优势来自选择了更适合高速移动信道的变换域。
频率选择性衰落主要由多径时延造成，而 OTFS 在延迟维度中直接表示这些路径；
高速移动造成的多普勒频移则在多普勒维度中直接表示。由于实际无线传播通常只由
有限数量的主要路径决定，延迟-多普勒域信道往往是稀疏的。接收机可以利用这种
稀疏结构做联合检测和干扰抵消，因此 OTFS 比传统 OFDM 更适合频率选择性衰落
和快速移动场景。

\section{结论}

本项目使用 MATLAB 完成了课程项目中要求的主要信号处理实验。实验结果说明，
时域反转和时间尺度变换会改变信号的时间结构，并在频域中体现为相位变化或
频率轴伸缩；幅度谱和相位谱分别携带不同类型的信息，二者缺一都会造成重构结果
与原信号明显不同；频域理想低通滤波可以通过 FFT、频域掩膜和 IFFT 直接实现。
第二部分的 OTFS 论文阅读进一步说明，选择合适的变换域可以让复杂信道呈现出
更清晰、更稀疏的结构，从而改善高速移动和频率选择性衰落场景下的通信性能。
